# **1. 数据结构：二叉树遍历的非递归实现**

**问题**：写出二叉树先序遍历的非递归算法，并分析其时间复杂度。
**考点**：栈的应用、遍历算法的灵活实现。

**回答**：

```python
def preorderTraversal(root):
    stack, result = [], []
    if root:
        stack.append(root)
    while stack:
        node = stack.pop()
        result.append(node.val)
        if node.right:  # 右子节点先入栈
            stack.append(node.right)
        if node.left:   # 左子节点后入栈
            stack.append(node.left)
    return result
```

**时间复杂度**：O(n)，其中n为节点数。每个节点入栈出栈各一次。
